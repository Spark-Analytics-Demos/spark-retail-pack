# Diagrams

This folder holds visual diagrams that support the design documents — architecture diagrams, ERDs, data flow charts, sequence diagrams, etc.

## Preferred formats

- **Mermaid** (`.mmd` or embedded in markdown) — version-controllable, renders in GitHub, modern editors, and most markdown viewers
- **Draw.io / diagrams.net** (`.drawio`) — for more complex visuals; can be exported to SVG/PNG
- **PNG / SVG** — for final rendered versions when needed for sales decks

## Why diagrams live here, not inline

For simple flowcharts, embed them directly in the relevant section using Mermaid code blocks. For larger or reused diagrams, store the source file here and reference it.
